A 3D endless runner mobile game built with Three.js. Navigate your hover-ship through a neon void, dodge obstacles, collect power-ups, and defeat bosses!

## Features

- **3D Graphics**: Built with Three.js for smooth 3D rendering
- **Mobile-First**: Touch controls optimized for mobile devices
- **Procedural Generation**: Infinite scrolling with randomly generated obstacles
- **Power-Up System**: Shield, Magnet, Speed Boost, Weapon, Score Multiplier
- **Roguelike Progression**: Permanent upgrades using collected crystals
- **Boss Battles**: Face epic bosses every 1000 meters
- **PWA Ready**: Install on your phone for offline play
- **Synthwave Aesthetic**: Neon cyberpunk visuals with particle effects

## Controls

### Mobile
- **Swipe Left/Right**: Change lanes
- **Swipe Up**: Jump over obstacles
- **Swipe Down**: Duck under obstacles
- **Tilt**: Fine movement control (optional in settings)

### Desktop
- **Arrow Keys / WASD**: Move and jump
- **Space**: Jump
- **P**: Pause

## How to Play

1. Navigate through three lanes avoiding obstacles
2. Collect crystals (ðŸ’Ž) to earn currency
3. Grab power-ups for temporary abilities
4. Build combos by narrowly avoiding obstacles
5. Defeat bosses to earn big crystal rewards
6. Use crystals to purchase permanent upgrades and new ships

## Deployment

Deployed on Vercel as a static site:
```bash
npx vercel --yes
```

## Tech Stack

- Three.js (3D rendering)
- Web Audio API (sound effects)
- LocalStorage (save data)
- Service Workers (PWA support)
- Vanilla JavaScript (no frameworks)
